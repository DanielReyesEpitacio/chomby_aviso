Datos personales que recopilamos

Para ofrecerte un servicio óptimo, recopilamos los siguientes datos personales:

  • Información de geolocalización: para brindarte información sobre la proximidad de las unidades de transporte a tu ubicación.
  • Datos de contacto: como tu dirección de correo electrónico.
  • Información sobre el uso de la aplicación: datos relacionados con tu interacción con la aplicación para mejorar nuestros servicios.

Finalidades del tratamiento de datos

Los datos que recopilamos serán utilizados para las siguientes finalidades:

  1. Proporcionar información en tiempo real sobre las rutas y medios de transporte disponibles.
  2. Informar sobre el tiempo estimado de viaje y costo del pasaje.
  3. Mejorar la experiencia del usuario mediante la personalización de la información.
  4. Realizar análisis estadísticos para optimizar el servicio.

Opciones y medios para limitar el uso o divulgación de datos personales

Puedes limitar el uso de tus datos personales a través de las configuraciones de tu dispositivo móvil, donde podrás desactivar la geolocalización o modificar los permisos de acceso a la aplicación.
